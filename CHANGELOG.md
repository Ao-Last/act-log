# 更新日志 (Changelog)

## [Unreleased]

### 新增 (Added)

#### 🌍 多语言支持 (i18n)
- 使用 i18next + react-i18next 实现完整的多语言系统
- 内置中文（简体）和英文两种语言
- 自动检测系统语言，智能选择默认语言
- **系统托盘菜单添加语言切换选项**，支持：
  - **自动（跟随系统）** - 根据系统语言自动选择
  - **中文** - 简体中文
  - **English** - 英文
- 在日志查看器右上角也可以切换语言（两个地方都可以切换）
- **托盘菜单根据当前语言显示**，不再是中英双语混合
- 支持实时切换语言，无需重启应用
- 语言偏好自动持久化保存到 electron-store
- 所有用户界面文本完整翻译：
  - **系统托盘菜单**（新增）
  - 日志查看器界面
  - 快速输入弹窗
  - 时间格式本地化
  - 日期格式本地化
  - 主题名称本地化（新增）
- 架构设计便于扩展更多语言

#### 技术实现
- 添加依赖：`i18next` 和 `react-i18next`
- 创建了清晰的翻译文件结构（`src/renderer/src/locales/`）
- 主进程添加语言翻译映射和管理逻辑
- 实现了主进程和渲染进程之间的语言同步：
  - `get-language` - 获取当前语言
  - `set-language` - 设置语言
  - `language-changed` - 语言变更事件通知
- 语言设置存储在 electron-store 中，支持 'auto'、'zh'、'en' 三种模式
- 新增 LanguageSwitcher 组件，提供优雅的语言切换 UI
- 所有组件（包括主进程）完全支持 i18n
- 主题名称根据当前语言显示

#### 文档
- 新增 `docs/I18N_GUIDE.md` - 完整的多语言支持指南
- 包含：配置说明、使用方法、添加新语言的步骤、最佳实践
- 更新 README.md，添加多语言支持章节和托盘菜单说明

#### 🎨 主题配置系统
- 添加了 6 种精美的预设主题色：靛蓝、蓝色、翡翠绿、玫瑰红、紫罗兰、琥珀色
- 系统托盘菜单新增"主题 (Theme)"选项，支持实时切换
- 主题选择会自动持久化保存，重启应用后保持上次选择
- 所有界面元素（托盘图标、弹窗、日志查看器）自动应用主题色

#### 技术实现
- 使用 `electron-store` 实现主题配置的持久化存储
- 添加了完整的 IPC 通信机制：
  - `get-theme` - 获取当前主题
  - `get-available-themes` - 获取所有可用主题
  - `theme-changed` - 主题变更事件通知
- 使用 CSS 变量实现动态主题切换
- 更新托盘图标生成逻辑，支持动态主题色

#### 文档
- 新增 `docs/THEME_CONFIG.md` - 主题配置详细说明
- 新增 `docs/THEME_COLORS.md` - 主题色彩对比表
- 更新 `README.md`，添加主题配置章节

### 改进 (Improved)
- 优化了 Popup 组件的主题应用逻辑
- 优化了 LogViewer 组件的主题应用逻辑
- 改进了主题切换的用户体验，支持实时预览

### 技术细节
- 新增依赖：`electron-store@^8.x`
- 更新了 TypeScript 类型定义，添加 Theme 接口
- 重构了托盘图标生成函数，使用主题配色
- 所有主题色遵循 Tailwind CSS v4 色彩规范
